# Brugada Syndrome Detection using ECG (V1–V3)

## Overview
Machine learning pipeline for detecting Brugada syndrome using ECG signals with interpretable feature engineering.

## Method
- WFDB ECG loading
- NeuroKit2 R-peak detection
- Feature engineering (HR, RR, QRS, ST, morphology)
- Classical ML models (Logistic Regression, Random Forest, XGBoost)

## Results
Best ROC-AUC: ~0.80

## Features
- V1–V3 ECG leads
- 24 engineered features per patient
